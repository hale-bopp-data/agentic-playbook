# FAQ: What is GEDI?

GEDI (Guardian EasyWay Delle Intenzioni) is a philosophy guardian agent
that evaluates decisions against 19 principles using an OODA loop.

5 themes: Craft & Quality, Action & Pragmatism, Risk & Resilience,
Security & Defense, Vision & Courage.

Key traits: non-blocking, principled, auditable, RAG-enhanced.

```bash
bash scripts/connections/gedi.sh consult "Should we skip code review?"
```
