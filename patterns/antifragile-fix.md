# Pattern: Antifragile Fix

## Problem

Fixes that only address symptoms recur.

## Solution

Every fix must leave the system more robust than before:

1. Root cause analysis — understand WHY
2. Structural fix — address the cause
3. Guardrail — prevent recurrence permanently
4. Documentation — record the fix

## Test

After the fix: is the system more robust? If no, the fix is incomplete.
