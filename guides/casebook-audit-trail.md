# Casebook Audit Trail

## What is a Casebook?

A casebook records every significant decision made by an advisory agent.
It serves as audit trail, precedent database, and training data.

## Format

```markdown
### Case #73 — S121: Quality gate decision

**Date**: 2026-03-10
**Query**: Should we publish with current quality?
**OODA**:
- Observe: Quality improved but not measured
- Orient: P01 (Quality) — ship when quality is proven
- Decide: CAUTION — measure before publishing
- Act: Run evaluation first

**Verdict**: CAUTION
**Principles**: P01, P08
```

## Auto-documentation Rule

After every consultation, document the case immediately.
